## 7. 游戏流程

### 7.1 主流程
1. 玩家进入游戏 → 显示主菜单
2. 选择场景 → 选择角色
3. 开始游戏 → 3秒倒计时
4. 游戏进行 → 双方对战
5. 一方血量为0 → 显示结束界面
6. 选择重新开始或返回主菜单

### 7.2 战斗流程
1. 玩家控制机甲移动
2. 选择攻击方式（轻攻击、重攻击、技能）
3. 攻击命中 → 对方受到伤害
4. 攻击未命中 → 无效果
5. 防御成功 → 抵挡伤害
6. 能量积累 → 释放技能
7. 连击达成 → 伤害加成

## 8. 技术实现

### 8.1 核心技术
- HTML5 Canvas 2D渲染
- JavaScript ES6+ 模块化
- 面向对象编程
- 资源预加载
- 性能优化

### 8.2 关键算法
- 碰撞检测：轴对齐边界框(AABB)算法
- 动画系统：帧动画系统
- 特效系统：粒子系统
- 物理系统：简单的重力和碰撞响应
- AI系统：基础的电脑AI

### 8.3 性能优化
- 精灵表批量渲染
- 离屏Canvas缓存
- 对象池复用
- 帧率控制
- 资源懒加载
